# Spot Covert Feature

he Spot Convert feature helps users swap their assets on the exchange. The following pairs are supported:

| Pair      | Convert Fee |
| --------- | ----------- |
| ETH-USDC  | 0.10%       |
| WBTC-USDC | 0.10%       |
| USDT-USDC | 0.01%       |



The trade happens at the **index price** of the asset.
